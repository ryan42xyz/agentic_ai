# Personal Company

Purpose:
Design and operate a personal system that compounds cognition, preserves optionality,
and reduces repeated thinking under uncertainty.

Core Principles:
1. Assets > Tasks
2. Short feedback loops first
3. Irreversible actions require explicit review

Operating Rule:
If a thought cannot become a reusable asset, it does not justify deep investment.

This repository is not a diary.
It is an operating system.
